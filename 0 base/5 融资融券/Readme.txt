融资融券利息费用:

    证券公司一般在金融机构六个月期贷款基准利率的基础上，上浮3个点，而融券费率一般高于融资利率。目前，已开展融资融券业务的11家券商中，最普遍的标准是：融资利率8.10%，融券费率10.10%

    融资利息：融资利息=融资金额×（融资利息/360）×实际使用天数

       （融资金额=融资买入成交价格*成交股数+交易费用）

    融券费用：融券费用=融券金额×（融券费率/360）×实际使用天数

    逾期利息：逾期利息=融资金额或融券金额×（逾期利息率/360）×实际逾期天数

. 关于融资融券的 操作 要点:

  万万切记:     

      融资: 

          1. 股票必须是符合当前 政策导向 ， 或行业 发展良好， 最好是非周期行业

          2. 股票跌幅据最近一波高点至少30%以上. 最好是50%以上

          3. 并且没有跌破 上一波 (高点-低点)X 61.8%.  这个可以是周线. 很关键.

          4. 横盘盘整时间: 至少21-34个交易日

          5. 投资一只股票初期, 绝对不能融资。

          6. 融资融券的交易成本, 除了利息之外, 还有交易费用, 最低是万六, 根本降不下来. 比普通账户成本高.

          7. 最最关键的是, 如果被套住, 为了避免平仓, 就要加仓, 但是这种股票一般都是破位下跌， 越补越跌，直到企稳. 而这个过程一般至少34个交易日, 大致2-3个月,甚至半年以上. 那么利息费用非常可怕. 而且合约一般都是半年, 逼的

          8. 融资进入位置, 最好是 接近历史底部位置.  安全第一.

     融券:

. 关于融资融券的信息披露

      各券商必须在每个交易日的22点前向沪深交易所 报送 当日融资融券交易的各项数据, 包括 融资买入额、融资还款额、融资余额、融券卖出量、融券余量等. 

      交易所 在次日开市前 对数据进行汇总, 在达到一定条件后予以 公开披露或暂停该品种 融资融券交易的措施. 

      具体规则如下:

      1. 单只标的证券的融资余额(或 融券余量) >= 该证券上市可流通市值(或 该证券上市可流通量) 20%时, 次日 开市前 像市场发布提示公告;

      2. 单只标的证券融资余额(或 融券余量) >= 该证券上市可流通市值(或该证券上市可流通量) 25%时, 次日 暂停该品种的融资或融券交易, 并在交易所网站 披露该品种的融资余额(或融券余量) 前五位券商名称及相应数据, 直到该比例降至20%以下;

      3. 单只标的证券 当日融资买入数量(或融券卖出数量)达到当日该品种总交易量的50%时, 次日开始前公布融资买入、融券卖出金额最大的五家券商名称。
